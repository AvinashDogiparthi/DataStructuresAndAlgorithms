## Problem Statement

Given two strings `s` and `t`, return **true** if `t` is an anagram of `s`, and **false** otherwise.

---

## Examples

### Example 1:
**Input:**
```text
s = "anagram", t = "nagaram"
```

**Output:**

```text
true
```

## Example 2:

**Input:**
```text
 s = "rat", t = "car"
```


**Output:**
```text
false
```
